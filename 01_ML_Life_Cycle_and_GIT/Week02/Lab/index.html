Git Commands Reference
Quick reference for all commands used in this lab
git init

Initialize new Git repository in current folder

git status

Check current state of repository

git add <file>

Stage file for commit

git add .

Stage all changed files

git commit -m "msg"

Save staged changes with message

git log --oneline

View commit history (compact)

git remote add origin <url>

Connect to remote repository

git push -u origin main

Upload commits to remote

git pull origin main

Download changes from remote
