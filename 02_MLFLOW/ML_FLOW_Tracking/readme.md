à¹€à¸£à¸µà¸¢à¸™à¸£à¸¹à¹‰à¸à¸²à¸£à¸šà¸±à¸™à¸—à¸¶à¸ Metrics à¸£à¸°à¸«à¸§à¹ˆà¸²à¸‡à¸à¸²à¸£ Training

# à¸•à¸±à¹‰à¸‡à¸„à¹ˆà¸² MLflow

print("=" * 50)
print("Lab 2: Training Loop Tracking")
print("=" * 50)

# -------------------------------------------------
# à¸ˆà¸³à¸¥à¸­à¸‡à¸à¸²à¸£ Training Model
# -------------------------------------------------
def simulate_training(epochs, initial_loss=1.0, learning_rate=0.1):
    """
    à¸ˆà¸³à¸¥à¸­à¸‡ Loss à¸—à¸µà¹ˆà¸¥à¸”à¸¥à¸‡à¸£à¸°à¸«à¸§à¹ˆà¸²à¸‡à¸à¸²à¸£ Training
    """
    losses = []
    val_losses = []
    
    for epoch in range(epochs):
        # à¸ˆà¸³à¸¥à¸­à¸‡ Training Loss (à¸¥à¸”à¸¥à¸‡à¹€à¸£à¸·à¹ˆà¸­à¸¢à¹†)
        train_loss = initial_loss * np.exp(-learning_rate * epoch)
        train_loss += np.random.normal(0, 0.02)  # à¹€à¸à¸´à¹ˆà¸¡ noise à¹€à¸¥à¹‡à¸à¸™à¹‰à¸­à¸¢
        train_loss = max(0.01, train_loss)  # à¹„à¸¡à¹ˆà¹ƒà¸«à¹‰à¸•à¸´à¸”à¸¥à¸š
        
        # à¸ˆà¸³à¸¥à¸­à¸‡ Validation Loss (à¸¥à¸”à¸¥à¸‡à¸Šà¹‰à¸²à¸à¸§à¹ˆà¸² Train)
        val_loss = initial_loss * np.exp(-learning_rate * 0.8 * epoch)
        val_loss += np.random.normal(0, 0.03)
        val_loss = max(0.02, val_loss)
        
        losses.append(train_loss)
        val_losses.append(val_loss)
    
    return losses, val_losses

# Parameters
EPOCHS = 50
LEARNING_RATE = 0.1
INITIAL_LOSS = 1.0

# à¹€à¸£à¸´à¹ˆà¸¡ Run
with mlflow.start_run(run_name="training-simulation"):
    
    # à¸šà¸±à¸™à¸—à¸¶à¸ Parameters
    print("\nğŸ“‹ à¸šà¸±à¸™à¸—à¸¶à¸ Parameters...")
    mlflow.log_params({
        "epochs": EPOCHS,
        "learning_rate": LEARNING_RATE,
        "initial_loss": INITIAL_LOSS,
        "optimizer": "SGD",
        "batch_size": 32
    })
    
    # à¸ˆà¸³à¸¥à¸­à¸‡à¸à¸²à¸£ Training
    print(f"\nğŸƒ à¹€à¸£à¸´à¹ˆà¸¡ Training à¸ˆà¸³à¸¥à¸­à¸‡ {EPOCHS} epochs...")
    train_losses, val_losses = simulate_training(
        EPOCHS, INITIAL_LOSS, LEARNING_RATE
    )
    
    # à¸šà¸±à¸™à¸—à¸¶à¸ Metrics à¸—à¸¸à¸ Epoch
    print("\nğŸ“Š à¸šà¸±à¸™à¸—à¸¶à¸ Metrics...")
    for epoch in range(EPOCHS):
        # à¸šà¸±à¸™à¸—à¸¶à¸ Loss à¸à¸£à¹‰à¸­à¸¡ step
        mlflow.log_metric("train_loss", train_losses[epoch], step=epoch)
        mlflow.log_metric("val_loss", val_losses[epoch], step=epoch)
        
        # à¸šà¸±à¸™à¸—à¸¶à¸ Accuracy à¸ˆà¸³à¸¥à¸­à¸‡ (à¹€à¸à¸´à¹ˆà¸¡à¸‚à¸¶à¹‰à¸™à¹€à¸£à¸·à¹ˆà¸­à¸¢à¹†)
        train_acc = 0.5 + 0.45 * (1 - np.exp(-0.1 * epoch))
        val_acc = 0.5 + 0.40 * (1 - np.exp(-0.1 * epoch))
        
        mlflow.log_metric("train_accuracy", train_acc, step=epoch)
        mlflow.log_metric("val_accuracy", val_acc, step=epoch)
        
        # à¹à¸ªà¸”à¸‡ Progress à¸—à¸¸à¸ 10 epochs
        if (epoch + 1) % 10 == 0:
            print(f"   Epoch {epoch+1}/{EPOCHS}: "
                  f"train_loss={train_losses[epoch]:.4f}, "
                  f"val_loss={val_losses[epoch]:.4f}")
    
    # à¸šà¸±à¸™à¸—à¸¶à¸ Final Metrics
    mlflow.log_metrics({
        "final_train_loss": train_losses[-1],
        "final_val_loss": val_losses[-1],
        "final_train_accuracy": train_acc,
        "final_val_accuracy": val_acc,
        "best_val_loss": min(val_losses),
        "best_epoch": val_losses.index(min(val_losses))
    })
    
    # -------------------------------------------------
    # à¸ªà¸£à¹‰à¸²à¸‡à¸à¸£à¸²à¸Ÿ Loss Curve à¹à¸¥à¸°à¸šà¸±à¸™à¸—à¸¶à¸à¹€à¸›à¹‡à¸™ Artifact
    # -------------------------------------------------
    print("\nğŸ“ˆ à¸ªà¸£à¹‰à¸²à¸‡à¸à¸£à¸²à¸Ÿ Loss Curve...")
    os.makedirs("outputs", exist_ok=True)
    
    fig, axes = plt.subplots(1, 2, figsize=(14, 5))
    
    # à¸à¸£à¸²à¸Ÿ Loss
    axes[0].plot(train_losses, label='Training Loss', color='blue')
    axes[0].plot(val_losses, label='Validation Loss', color='orange')
    axes[0].set_xlabel('Epoch')
    axes[0].set_ylabel('Loss')
    axes[0].set_title('Loss Curve')
    axes[0].legend()
    axes[0].grid(True, alpha=0.3)
    
    # à¸à¸£à¸²à¸Ÿ Accuracy
    train_accs = [0.5 + 0.45 * (1 - np.exp(-0.1 * e)) for e in range(EPOCHS)]
    val_accs = [0.5 + 0.40 * (1 - np.exp(-0.1 * e)) for e in range(EPOCHS)]
    
    axes[1].plot(train_accs, label='Training Accuracy', color='blue')
    axes[1].plot(val_accs, label='Validation Accuracy', color='orange')
    axes[1].set_xlabel('Epoch')
    axes[1].set_ylabel('Accuracy')
    axes[1].set_title('Accuracy Curve')
    axes[1].legend()
    axes[1].grid(True, alpha=0.3)
    
    plt.tight_layout()
    plt.savefig("outputs/training_curves.png", dpi=150)
    plt.close()
    
    # à¸šà¸±à¸™à¸—à¸¶à¸à¸à¸£à¸²à¸Ÿà¹€à¸›à¹‡à¸™ Artifact
    mlflow.log_artifact("outputs/training_curves.png")
    print("   âœ… à¸šà¸±à¸™à¸—à¸¶à¸à¸à¸£à¸²à¸Ÿ training_curves.png")

print("\n" + "=" * 50)
print("ğŸ‰ Lab 2 à¹€à¸ªà¸£à¹‡à¸ˆà¸ªà¸´à¹‰à¸™!")
print("=" * 50)
print(f"\nğŸ“Š à¸”à¸¹à¸œà¸¥à¸¥à¸±à¸à¸˜à¹Œà¹„à¸”à¹‰à¸—à¸µà¹ˆ: http://127.0.0.1:8080")
print(f"   â†’ à¹€à¸¥à¸·à¸­à¸ Experiment: lab2-training-loop")
print(f"   â†’ à¸„à¸¥à¸´à¸à¸—à¸µà¹ˆ Run à¹à¸¥à¹‰à¸§à¹„à¸›à¸—à¸µà¹ˆ tab 'Charts'")
print(f"   â†’ à¸ˆà¸°à¹€à¸«à¹‡à¸™ Loss Curve à¹à¸¥à¸° Accuracy Curve")

"""
Lab 4: Hyperparameter Comparison
================================
à¹€à¸£à¸µà¸¢à¸™à¸£à¸¹à¹‰à¸à¸²à¸£à¸—à¸”à¸¥à¸­à¸‡ Hyperparameters à¸«à¸¥à¸²à¸¢à¸„à¹ˆà¸²à¹à¸¥à¸°à¹€à¸›à¸£à¸µà¸¢à¸šà¹€à¸—à¸µà¸¢à¸šà¸œà¸¥
"""

import mlflow
from sklearn.datasets import load_breast_cancer
from sklearn.model_selection import train_test_split, cross_val_score
from sklearn.ensemble import RandomForestClassifier
from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score
import warnings
warnings.filterwarnings('ignore')

mlflow.set_tracking_uri("http://127.0.0.1:8080")
mlflow.set_experiment("lab4-hyperparameter-tuning")

print("=" * 50)
print("Lab 4: Hyperparameter Comparison")
print("=" * 50)

# à¹‚à¸«à¸¥à¸”à¸‚à¹‰à¸­à¸¡à¸¹à¸¥
data = load_breast_cancer()
X_train, X_test, y_train, y_test = train_test_split(
    data.data, data.target, test_size=0.2, random_state=42
)

# à¸à¸³à¸«à¸™à¸” Hyperparameter Search Space
param_grid = {
    'n_estimators': [50, 100, 200],
    'max_depth': [3, 5, 10, None],
    'min_samples_split': [2, 5, 10]
}

total_experiments = (len(param_grid['n_estimators']) * 
                    len(param_grid['max_depth']) * 
                    len(param_grid['min_samples_split']))

print(f"\nğŸ“‹ Total experiments: {total_experiments}")

best_accuracy = 0
best_params = None
experiment_count = 0

for n_est in param_grid['n_estimators']:
    for max_d in param_grid['max_depth']:
        for min_split in param_grid['min_samples_split']:
            experiment_count += 1
            
            max_d_str = str(max_d) if max_d is not None else "None"
            run_name = f"rf_nest{n_est}_depth{max_d_str}_split{min_split}"
            
            with mlflow.start_run(run_name=run_name):
                mlflow.log_params({
                    "n_estimators": n_est,
                    "max_depth": max_d if max_d is not None else "unlimited",
                    "min_samples_split": min_split,
                    "random_state": 42
                })
                
                model = RandomForestClassifier(
                    n_estimators=n_est,
                    max_depth=max_d,
                    min_samples_split=min_split,
                    random_state=42,
                    n_jobs=-1
                )
                model.fit(X_train, y_train)
                
                predictions = model.predict(X_test)
                
                accuracy = accuracy_score(y_test, predictions)
                precision = precision_score(y_test, predictions)
                recall = recall_score(y_test, predictions)
                f1 = f1_score(y_test, predictions)
                
                cv_scores = cross_val_score(model, X_train, y_train, cv=5)
                
                mlflow.log_metrics({
                    "accuracy": accuracy,
                    "precision": precision,
                    "recall": recall,
                    "f1_score": f1,
                    "cv_mean": cv_scores.mean(),
                    "cv_std": cv_scores.std()
                })
                
                if accuracy > best_accuracy:
                    best_accuracy = accuracy
                    best_params = {
                        "n_estimators": n_est,
                        "max_depth": max_d,
                        "min_samples_split": min_split
                    }
                    mlflow.set_tag("is_best", "true")
                else:
                    mlflow.set_tag("is_best", "false")
                
                print(f"   [{experiment_count}/{total_experiments}] "
                      f"{run_name}: accuracy={accuracy:.4f}")

print("\n" + "=" * 50)
print(f"ğŸ† Best Accuracy: {best_accuracy:.4f}")
print(f"   Best Params: {best_params}")
print("=" * 50)
