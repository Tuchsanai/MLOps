# 🚀 Mini Project: End-to-End AI Application

## From Idea to Production — พัฒนาตัวคุณสู่ ML Engineer มืออาชีพ

---

## 🎯 วัตถุประสงค์

> *"ไม่ใช่แค่โค้ดที่รันได้ แต่คือระบบที่ใช้งานได้จริงและสร้าง Impact"*

Mini Project นี้ออกแบบให้นักศึกษาได้สัมผัสประสบการณ์การพัฒนา **AI Application แบบ End-to-End** เสมือนทำงานใน Industry จริง — ตั้งแต่ตั้งโจทย์ ออกแบบระบบ พัฒนา ทดสอบ จนถึง Deploy ให้ผู้ใช้ทั่วโลกเข้าถึงได้

### 🔄 ML Lifecycle ที่ต้องครอบคลุม

```
💡 Problem → 🎯 Solution → 👥 Users → ⚙️ Development → 🚀 Deployment → 📈 Impact
```

### ⚠️ ข้อกำหนดพิเศษที่ต้องทราบ

Mini Project ในรายวิชานี้มีความพิเศษกว่าปกติ เนื่องจากเป็นการ **ควบรวมเนื้อหาจาก 2 รายวิชา** ได้แก่

| รายวิชา | ลักษณะเนื้อหา |
|---------|--------------|
| **TEAM-PROJECT 3** | การทำงานเป็นทีม การบริหารโครงการ และการนำเสนอผลงาน |
| **MACHINE LEARNING OPERATIONS** | การพัฒนาระบบ ML แบบครบวงจร ตั้งแต่ Development จนถึง Production |

ด้วยเหตุนี้ **คุณภาพของงานจึงต้องใกล้เคียงหรือเทียบเท่ากับ Project จบ** ซึ่งหมายความว่างานต้องมีความสมบูรณ์ทั้งในแง่เทคนิค การออกแบบ และการนำไปใช้งานจริง


### 📌 หลักเกณฑ์การเลือกหัวข้อ

นักศึกษา AIT มีอิสระในการเลือกหัวข้อโปรเจค โดยหัวข้อที่เลือกต้องสอดคล้องกับ **เป้าหมายการพัฒนาที่ยั่งยืน (Sustainable Development Goals: SDGs)** ขององค์การสหประชาชาติ ซึ่งเป็นกรอบการพัฒนาที่ทั่วโลกให้ความสำคัญ


### 🌍 เป้าหมายการพัฒนาที่ยั่งยืน 17 ข้อ (The 17 Goals)

#### 🔴 กลุ่มที่ 1: การขจัดความยากจนและความหิวโหย

| SDG | ชื่อเป้าหมาย | รายละเอียด |
|:---:|-------------|-----------|
| **1** | **ยุติความยากจน** | ขจัดความยากจนทุกรูปแบบในทุกพื้นที่ทั่วโลก |
| **2** | **ยุติความหิวโหย** | สร้างความมั่นคงทางอาหาร ยกระดับโภชนาการ และส่งเสริมเกษตรกรรมที่ยั่งยืน |

#### 🟠 กลุ่มที่ 2: สุขภาพและการศึกษา

| SDG | ชื่อเป้าหมาย | รายละเอียด |
|:---:|-------------|-----------|
| **3** | **สุขภาพและความเป็นอยู่ที่ดี** | สร้างหลักประกันการมีสุขภาวะที่ดีสำหรับทุกคนในทุกช่วงวัย |
| **4** | **การศึกษาที่มีคุณภาพ** | สร้างการศึกษาที่ครอบคลุม เท่าเทียม และสนับสนุนการเรียนรู้ตลอดชีวิต |

#### 🟡 กลุ่มที่ 3: ความเสมอภาคและโครงสร้างพื้นฐาน

| SDG | ชื่อเป้าหมาย | รายละเอียด |
|:---:|-------------|-----------|
| **5** | **ความเสมอภาคทางเพศ** | บรรลุความเท่าเทียมระหว่างเพศ และเพิ่มบทบาทของสตรีและเด็กหญิง |
| **6** | **น้ำสะอาดและสุขาภิบาล** | จัดการน้ำและระบบสุขาภิบาลอย่างยั่งยืนสำหรับทุกคน |
| **7** | **พลังงานสะอาด** | สร้างการเข้าถึงพลังงานสมัยใหม่ในราคาที่เอื้อมถึงและยั่งยืน |

#### 🟢 กลุ่มที่ 4: เศรษฐกิจและอุตสาหกรรม

| SDG | ชื่อเป้าหมาย | รายละเอียด |
|:---:|-------------|-----------|
| **8** | **งานที่มีคุณค่าและเศรษฐกิจที่เติบโต** | ส่งเสริมการเติบโตทางเศรษฐกิจที่ยั่งยืน การจ้างงานเต็มที่ และงานที่มีคุณค่า |
| **9** | **อุตสาหกรรม นวัตกรรม และโครงสร้างพื้นฐาน** | สร้างโครงสร้างพื้นฐานที่ยืดหยุ่น ส่งเสริมอุตสาหกรรมและนวัตกรรมที่ยั่งยืน |
| **10** | **ลดความเหลื่อมล้ำ** | ลดความไม่เสมอภาคทั้งภายในประเทศและระหว่างประเทศ |

#### 🔵 กลุ่มที่ 5: เมืองและการบริโภคที่ยั่งยืน

| SDG | ชื่อเป้าหมาย | รายละเอียด |
|:---:|-------------|-----------|
| **11** | **เมืองและชุมชนที่ยั่งยืน** | สร้างเมืองและการตั้งถิ่นฐานที่ครอบคลุม ปลอดภัย และยั่งยืน |
| **12** | **การผลิตและบริโภคที่ยั่งยืน** | สร้างแบบแผนการผลิตและการบริโภคที่รับผิดชอบต่อสิ่งแวดล้อม |

#### 🟣 กลุ่มที่ 6: สิ่งแวดล้อมและทรัพยากรธรรมชาติ

| SDG | ชื่อเป้าหมาย | รายละเอียด |
|:---:|-------------|-----------|
| **13** | **รับมือการเปลี่ยนแปลงสภาพภูมิอากาศ** | ดำเนินการอย่างเร่งด่วนเพื่อต่อสู้กับ Climate Change และผลกระทบ |
| **14** | **ทรัพยากรทางทะเล** | อนุรักษ์และใช้ประโยชน์จากมหาสมุทร ทะเล และทรัพยากรทางทะเลอย่างยั่งยืน |
| **15** | **ระบบนิเวศบนบก** | ปกป้องและฟื้นฟูระบบนิเวศบนบก จัดการป่าไม้ ต่อสู้กับการกลายเป็นทะเลทราย และหยุดการสูญเสียความหลากหลายทางชีวภาพ |

#### ⚪ กลุ่มที่ 7: สันติภาพและความร่วมมือ

| SDG | ชื่อเป้าหมาย | รายละเอียด |
|:---:|-------------|-----------|
| **16** | **สันติภาพและความยุติธรรม** | ส่งเสริมสังคมที่สงบสุข ให้ทุกคนเข้าถึงความยุติธรรม และสร้างสถาบันที่มีประสิทธิภาพ |
| **17** | **ความร่วมมือเพื่อการพัฒนาที่ยั่งยืน** | เสริมสร้างความร่วมมือระดับโลกเพื่อบรรลุเป้าหมายการพัฒนาที่ยั่งยืน |

## 📋 โครงสร้างการนำเสนอ (7 องค์ประกอบหลัก)

### 1️⃣ Problem Statement — ปัญหาที่เราจะแก้คืออะไร

**🎯 เป้าหมาย:** ทำให้ผู้ฟังรู้สึกว่า นี่คือปัญหาที่สำคัญ ถ้าไม่ได้รับการแก้ไข จะส่งผลร้ายต่อสังคม และที่สำคัญ ต้องทำให้เชื่อว่า ปัญหานี้มีอยู่จริง ไม่ได้คิดเอาเอง แต่มีข้อมูลรองรับ

**📝 สิ่งที่ต้องมี:**
- 📊 **ข้อมูลสนับสนุน:** สถิติ, บทสัมภาษณ์, Insights, Case Studies
- 🎭 **Persona/Story:** สร้างตัวละครที่เผชิญปัญหานี้ ร้อยเรียงเรื่องราว
- 🔍 **Root Cause Analysis:** วิเคราะห์สาเหตุที่แท้จริงของปัญหา
- 💥 **Impact of Problem:** ผลกระทบหากไม่แก้ไข (ต่อบุคคล/สังคม/เศรษฐกิจ)

**💡 Tips:**
- ปัญหาต้องพิสูจน์ได้ว่ามีอยู่จริง — ไม่ใช่ assumption ที่คิดเอาเอง
- หากปัญหาซับซ้อน ต้องหาวิธีอธิบายให้คนทั่วไปเข้าใจง่าย
- ศึกษา Pitch Deck ของ Startup ที่แก้ปัญหาใกล้เคียง
- ยิ่งมีประสบการณ์ตรงหรือลงพื้นที่สัมผัสปัญหายิ่งดี

---

### 2️⃣ Solution — สิ่งที่เราทำคืออะไร แก้ปัญหาได้อย่างไร

**🎯 เป้าหมาย:** ทำให้ผู้ฟังรู้สึกว่า Solution นี้ "ทำได้จริง" "แตกต่างจาก Solution อื่น" และ "แก้ปัญหาได้จริง"

**📝 สิ่งที่ต้องนำเสนอ:**
- 🧠 **แนวคิด:** ไอเดีย Solution นี้คืออะไร? ดำเนินการอย่างไร?
- 🖼️ **Prototype:** โชว์ภาพ/Demo ให้เห็นภาพชัดเจน
- ✅ **ประสิทธิผล:** แก้ปัญหาได้อย่างไร? ส่งผลดีอย่างไร?
- 🆚 **Differentiation:** ดีกว่าแบบอื่นอย่างไร?
- 💬 **Validation:** มี Feedback จากกลุ่มเป้าหมายหรือไม่?

**🔧 Technical Solution Components:**

```
┌─────────────────────────────────────────────────────────┐
│                    AI/ML Solution                        │
├─────────────────────────────────────────────────────────┤
│  📊 Data Pipeline    → ETL, Validation, Versioning      │
│  🧠 Model            → Training, Optimization, Tracking │
│  📈 Evaluation       → Metrics, A/B Testing             │
│  🌐 API/Interface    → FastAPI, Streamlit, Gradio       │
│  🐳 Deployment       → Docker, Cloud                    │
│  📡 Monitoring       → Performance, Drift Detection     │
└─────────────────────────────────────────────────────────┘
```

### 3️⃣ Target Users / Target Market — กลุ่มเป้าหมายคือใคร

**🎯 เป้าหมาย:** ทำให้ผู้ฟังรู้สึกว่า เราเข้าใจกลุ่มเป้าหมาย เข้าใจความต้องการ ระบุได้ว่าเป็นคนกลุ่มไหน มีจำนวนเท่าไร และมีช่องทางเข้าถึงได้

**📝 สิ่งที่ต้องนำเสนอ:**
- 👤 **User Persona:** รูปภาพ, ข้อมูล Demographics, Pain Points
- 📊 **Segmentation:** แบ่งกลุ่มผู้ใช้ตาม Characteristics
- 📏 **Market Size:** TAM/SAM/SOM หรือจำนวนผู้ใช้โดยประมาณ
- 🛤️ **Go-to-Market:** วิธีเข้าถึงกลุ่มเป้าหมาย (Partners, Channels)
- 🔍 **Competitive Analysis:** รู้จัก Solution อื่นที่ target กลุ่มใกล้เคียง

### 4️⃣ Social Impact & Assessment — ผลลัพธ์และการวัดผล

**🎯 เป้าหมาย:** ทำให้ผู้ฟังรู้สึกว่า จะมี Impact เกิดขึ้นจริง ช่วยสร้างการเปลี่ยนแปลงได้จริง และมีแนวทางในการวัดผลชัดเจน

**📝 Framework การวัดผล:**

| ระดับ | Metrics ตัวอย่าง |
|:-----:|-----------------|
| 🎯 **Output** | จำนวนผู้ใช้, Transactions, API Calls |
| 📈 **Outcome** | ประสิทธิภาพที่เพิ่มขึ้น, เวลาที่ประหยัด, ความแม่นยำ |
| 💫 **Impact** | การเปลี่ยนแปลงพฤติกรรม, คุณภาพชีวิต, ผลกระทบทางสังคม |

**📊 ML-Specific Metrics:**

| ประเภท | Metrics |
|--------|---------|
| **Classification** | Accuracy, Precision, Recall, F1, AUC-ROC |
| **Regression** | MAE, MSE, RMSE, R² |
| **NLP** | BLEU, ROUGE, Perplexity |
| **Computer Vision** | mAP, IoU, FID |
| **Recommendation** | Hit Rate, NDCG, Coverage |
| **Business** | Conversion Rate, Cost Savings, Time Saved |

### 5️⃣ Team — ทำไมทีมเราถึงเป็นทีมที่ดีที่สุด

### 6️⃣ Sustainability — ศักยภาพในการดำรงอยู่ระยะยาว

**📝 สิ่งที่ต้องนำเสนอ:**
- 💰 **Revenue Model:** โมเดลการสร้างรายได้ (ถ้ามี)
- 📊 **Cost Structure:** ค่าใช้จ่ายหลัก (Infra, API, Maintenance)

- 📅 **Roadmap:** แผนพัฒนาในอนาคต
- 🔄 **Scalability:** ขยายได้อย่างไรเมื่อผู้ใช้เพิ่ม
